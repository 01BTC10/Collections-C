Detailed Description

HashSet iterator object.

Used to iterate over the elements of the set. The iterator also supports operations for safely removing elements during iteration.

HashSetIter i;
hashset_iter_init(&i);
while (hashset_iter_has_next(&i)) {
MyType *e = hashset_iter_next(&i);
}
